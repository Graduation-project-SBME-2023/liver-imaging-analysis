Active learning

Active learning enhances medical image segmentation by combining human expertise and machine learning. MONAI Label, an intelligent open-source tool, facilitates the creation of annotated datasets and AI annotation models. Additionally, the Slicer application provides advanced image analysis capabilities, further optimizing segmentation accuracy.

Slicer

To perform annotations, it is important to obtain the slicer software. You can download it from this link. Additionally, you need to download an extension called "Monailabel" for the slicer

Download Slicer

How it Works

Open MonaiLabel

Begin the MonaiLabel extension and enter the IP address for the worldwide server to initiate.

Run The Model

To initiate the segmentation process, simply press the run button.

Annotate

To start the editing process, utilize this designated box, which provides you with a comprehensive set of tools required for effortless manual annotation.

Submit and fine-tune

Afterward, click on the "submit label" button to save the newly annotated mask. Next, proceed to train the model for one epoch, using the updated mask to fine-tune the model's performance.
